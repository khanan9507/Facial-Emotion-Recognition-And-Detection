# Facial Emotion Recognition And Detection

A real-time facial emotion detection system built using Deep Learning (CNN) and OpenCV.

## 🎯 Project Overview
This BTech AI/ML project detects 7 human emotions from facial expressions:
**Happy | Sad | Angry | Surprised | Fear | Disgust | Neutral**

## 🛠️ Tech Stack
- Python 3.x
- TensorFlow / Keras
- OpenCV
- FER-2013 Dataset (35,000+ images)
- Google Colab (GPU Training)

## 📊 Model Performance
- Architecture: Custom CNN (3 Conv layers)
- Training Epochs: 30
- Validation Accuracy: ~57%
- Confidence on test image: 85% (Happy)

## 🚀 How to Run
1. Open `facial_emotion_recognition.ipynb` in Google Colab
2. Run all cells step by step
3. Upload any face image to detect emotion

## 📁 Dataset
FER-2013 Dataset from Kaggle
- 35,887 grayscale images
- 7 emotion classes
- 48x48 pixel images
